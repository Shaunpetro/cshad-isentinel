**Privacy-first community safety app for South Africa**  
Built with React Native (Expo), Supabase, and TypeScript.

## 📱 Download & Install Beta APK

[![Download APK](https://img.shields.io/badge/Download-APK-brightgreen?style=for-the-badge&logo=android)](https://github.com/Shaunpetro/cshad-isentinel/releases/latest)

> Tap the badge above → find the `.apk` file under "Assets" → download to your Android device.

### Step‑by‑step installation

1. **Enable “Install from unknown sources”**  
   - Go to **Settings → Security** (or **Apps & notifications → Special app access → Install unknown apps**).  
   - Choose your browser or file manager and toggle **Allow from this source**.

2. **Download the APK**  
   - Tap the **Download APK** badge above and open the link on your phone.  
   - Scroll to the **Assets** section of the latest release and tap the `.apk` file.

3. **Install**  
   - Once downloaded, open the file (or tap the download notification).  
   - Tap **Install** and wait a few seconds.

4. **Open CSHAD iSentinel News**  
   - The app will appear in your app drawer.  
   - You may be asked to grant location, notification, or other permissions – we recommend allowing them for the best experience.

> **Troubleshooting**  
> - If the APK won’t install, double‑check that “Unknown sources” is enabled for the app you’re using to open the APK.  
> - If the link doesn’t work, visit `https://github.com/Shaunpetro/cshad-isentinel/releases` and download the latest `.apk` manually.

## 💬 Send Feedback

In the app, go to **Settings → Send Feedback**. You’ll be able to:

- Report a **bug**  
- Suggest a **feature**  
- Ask a **question**  

Your feedback opens your email client and is sent directly to the developer. No data is collected inside the app.

## 🚀 Features

- Real‑time community safety news (RSS feeds, tips)
- Interactive safety map with geolocation
- Anonymous tip submission
- Load shedding status & schedules (Eskom, free API)
- Weather alerts (OpenWeatherMap, free API)
- Developer credits & brand identity
