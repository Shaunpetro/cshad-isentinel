## 2. COMPLETED FEATURES & FIXES

### 2.1 Core App Fixes
- **OTA Updates** – Switched to Expo’s `ON_LOAD` policy; removed buggy manual OTA logic.  
- **App Hang on Startup** – Removed Onboarding screen; location permissions deferred to on‑demand.  
- **Logo & Branding** – All assets reference `cshad-isentinel-logo-main.png` consistently.  
- **TypeScript** – Zero errors; all barrel exports aligned.

### 2.2 Journalist Verification
- **Settings → Become a Journalist** opens the Vercel moderator registration page.  
- **Verified Journalist badge** appears on feed cards (`sourceType === 'journalist'` and `isVerified`).  
- **Journalist list** in Alerts tab fetches real `profiles` where `verified_journalist = true`.

### 2.3 Opportunities Tab (Main Feature)
- **Three top‑level tabs**: Tenders, Jobs, Bursaries.  
- **Glass‑morphism detail modal** with `expo-blur`, drag handle, swipe‑to‑dismiss, adaptive labels.  
- **Premium/Free simulation** with subscription toggle and upgrade modal (demo).  
- **Adaptive filters** via bottom sheet: Province, Industry/Field of Study, Submission Type (tenders only).  
- **Filters clear automatically** when switching tabs.  
- **Briefing info** displayed only for tenders; includes digital meeting links.  
- **Tender Number** (source_id) shown in modal for tenders; “Reference” for others.  
- **Direct document downloads** using `expo-file-system` and `expo-sharing`.  
- **Comprehensive detail display**: Advertised date, Closing date (with time), Submission type, Company, Location, Province, Briefing details.

### 2.4 Data Pipelines (Automated)
- **Tenders** – Puppeteer scraper for `etenders.gov.za` with pagination, retries, and full detail extraction (province, closing date, briefing, documents). 1,526 entries seeded.  
- **Jobs** – Adzuna API scraper (`f6dee8e7` / `bb7999107b22ce17e9c8eab1f6fceb76`) extracting province, city, subcategory, salary. 100+ live jobs.  
- **Bursaries** – Combined scraper for `bursaries.co.za` (paginated) and `allbursaries.co.za` (static cheerio) with deduplication. 200+ entries expected.  
- **Backfill scripts** for enriching existing data (province, dates, briefing, tender numbers).  
- **Daily GitHub Action** (`daily-opportunities.yml`) scheduled at 06:00 UTC (manual trigger also available).

### 2.5 Mobile App UX
- **Multi‑language support** (English, Afrikaans, isiZulu).  
- **Dark / Light / System theme**.  
- **Live News Feed**, Safety Map, Anonymous Tip, Load Shedding, Weather Alerts (existing from Beta 2).  
- **Push Notifications** (Firebase) with test button.  
- **Feedback Modal**, Developer Credits, Privacy Dashboard.  
- **Hazard Reporting** with live GPS, draggable pin, community voting.  
- **Bottom tab navigator** with 6 tabs (News, Map, Opportunities, Tip, Alerts, Settings).

## 3. DATABASE SCHEMA (Key Tables)

### `opportunities`
```sql
CREATE TABLE opportunities (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  title text NOT NULL,
  body text,
  category text NOT NULL,          -- 'tender', 'job', 'bursary', 'internship', 'learnership'
  subcategory text,
  province text,
  location_name text,
  latitude float8,
  longitude float8,
  closing_date timestamptz NOT NULL,
  date_advertised timestamptz,
  apply_url text,
  tender_docs jsonb DEFAULT '[]',
  salary_range text,
  company_name text,
  author_id uuid REFERENCES profiles(id),
  status text DEFAULT 'published', -- 'published', 'draft', 'closed'
  is_featured boolean DEFAULT false,
  is_premium boolean DEFAULT false,
  submission_type text,
  briefing_required boolean DEFAULT false,
  briefing_mandatory boolean DEFAULT false,
  briefing_details text,
  source_id text,                  -- deduplication key (e.g., 'adzuna-123', 'tender-ABC')
  created_at timestamptz DEFAULT now(),
  updated_at timestamptz DEFAULT now()
);
```

## 6. HOW TO START A NEW DEVELOPER / NEW CHAT

1. Clone `beta3` branch and run `npm install`.  
2. Set environment variables (`.env` with Supabase keys, Adzuna keys).  
3. Run `npx tsc --noEmit` to verify cleanliness.  
4. To test the mobile app:  
   ```powershell
   npx expo start --dev-client
   ```
5. To run a scraper locally:  
   ```powershell
   $env:SUPABASE_URL = "…"
   $env:SUPABASE_SERVICE_ROLE_KEY = "…"
   node scripts/scrape-<name>.js
   ```
6. To build for Play Store:  
   ```powershell
   npx eas build --platform android --profile production --non-interactive
   ```
