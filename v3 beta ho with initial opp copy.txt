
### 2.6 News Images
- **File:** `scripts/scrape-rss-feeds.js`
- **Fix:** In the `fetchAndInsertArticles` loop, try multiple sources for the image:
  1. `article.enclosure?.url`
  2. `article['media:content']?.url`
  3. `article['media:thumbnail']?.url`
  4. Regex extract first `<img>` from `article.content`
  5. Fallback: `null` (UI will show a placeholder)

## 3. DATABASE SCHEMA (Key Tables)

### `opportunities`
```sql
CREATE TABLE opportunities (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  title text NOT NULL,
  body text,
  category text NOT NULL,          -- 'tender', 'job', 'bursary', 'internship', 'learnership'
  subcategory text,
  province text,
  location_name text,
  latitude float8,
  longitude float8,
  closing_date timestamptz NOT NULL,
  date_advertised timestamptz,
  apply_url text,
  tender_docs jsonb DEFAULT '[]',
  salary_range text,
  company_name text,
  author_id uuid REFERENCES profiles(id),
  status text DEFAULT 'published', -- 'published', 'draft', 'closed'
  is_featured boolean DEFAULT false,
  is_premium boolean DEFAULT false,
  submission_type text,
  briefing_required boolean DEFAULT false,
  briefing_mandatory boolean DEFAULT false,
  briefing_details text,
  source_id text,                  -- deduplication key (e.g., 'adzuna-123', 'tender-ABC')
  created_at timestamptz DEFAULT now(),
  updated_at timestamptz DEFAULT now()
);
```

## 6. HOW TO START A NEW DEVELOPER / NEW CHAT

1. Clone `beta3` branch and run `npm install`.  
2. Set environment variables (`.env` with Supabase keys, Adzuna keys).  
3. Run `npx tsc --noEmit` to verify cleanliness.  
4. To test the mobile app:  
   ```powershell
   npx expo start --dev-client
   ```
5. To run a scraper locally:  
   ```powershell
   $env:SUPABASE_URL = "…"
   $env:SUPABASE_SERVICE_ROLE_KEY = "…"
   node scripts/scrape-<name>.js
   ```
6. To build for Play Store:  
   ```powershell
   npx eas build --platform android --profile production --non-interactive
   ```
